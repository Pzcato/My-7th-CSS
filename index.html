Here is some text that will be a paragraph. We can modify the text with CSS in different ways. We can use inline, embedding, and external styles. We can specify where the style goes by using class, id, and descendant selectors. Lastly, we should know the order of precedence.

The Order of Precedence

  1. Browser Defaults - basic colors, fonts, alignment, and other web page features that generate from the browser
  2. External Styles - coming from a .css file
  3. Embedded Styles - inserted in the head section of the html web page
  4. Inline Styles - inserted in the individual html tags/elements on a web page
